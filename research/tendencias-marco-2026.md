## O Momento Atual

### Vibe Coding: De Trend a Mainstream

O termo foi cunhado por **Andrej Karpathy** em fevereiro de 2025 num tweet com 4,5 milhões de visualizações. Hoje, em março de 2026:

- Foi **Palavra do Ano 2025** pelo Collins English Dictionary
- O mercado foi avaliado em **$2,96 bilhões** e deve atingir $12,3 bilhões em 2027
- **92% dos devs americanos** usam ferramentas de AI para código diariamente
- **60% de todo código novo** será AI-generated em 2026 (Gartner)

O debate mudou: não é mais "isso vai funcionar?" mas "até onde vai?" e "quais são os riscos reais?"

## Debates Quentes no Ecossistema (Twitter/X e Reddit)

### 1. "Vibe Coding funciona para produção?"
O grande debate do momento. Um estudo da CodeRabbit (dez/2025) analisou 470 PRs open-source e encontrou:
- Código AI-assisted tem **1,7x mais issues "major"**
- **2,74x mais vulnerabilidades de segurança**
- **75% mais misconfigurations**

Isso virou munição para críticos e conteúdo de resposta dos defensores.

**Ângulo de conteúdo**: "Os dados assustadores sobre código gerado por AI — e por que você precisa saber"

### 2. Agentes autônomos vs. AI assistida
A linha entre "usar AI para ajudar a codar" e "deixar AI codar enquanto você dorme" está sumindo. Claude Code rodando em modo autônomo é o exemplo mais avançado disso.

**Ângulo de conteúdo**: "Eu deixei o Claude Code trabalhar sozinho por 8 horas — o que aconteceu"

### 3. "Os programadores vão sumir?"
Yann LeCun e outros pesquisadores continuam defendendo que programadores ainda são necessários. Sam Altman sugere o contrário. O debate nunca para.

**Ângulo de conteúdo**: Formato debate/opinião forte

### 4. Microsoft unificou AutoGen + Semantic Kernel
Em outubro de 2025, a Microsoft mergiu o AutoGen com o Semantic Kernel num único framework. Disponível em GA em Q1 2026 (agora). Pouca cobertura em PT-BR.

**Ângulo de conteúdo**: "A Microsoft quer dominar o mercado de agentes — entenda o movimento"

## GitHub: Repositórios e Padrões para Observar

### Tipos de repos que viralizam
1. **Frameworks de agentes novos** — qualquer nova abstração para orquestrar LLMs
2. **Demos de Claude Code fazendo coisas impossíveis** — repos com vídeos de Claude codando autonomamente
3. **Alternativas open-source** a ferramentas pagas (Cursor, GitHub Copilot)
4. **Benchmarks e comparativos** — repos que testam modelos entre si

### Exemplos de repos relevantes recentes
- `wshobson/agents` — Intelligent automation for Claude Code
- `ruvnet/ruflo` — Agent orchestration platform for Claude
- `d60/twikit` — Twitter scraper sem API key
- `vladkens/twscrape` — Twitter API scraper with auth support

## Conteúdo que Está Performando (Padrões)

### No TikTok/Instagram Reels
- **"Eu pedi para a AI..."** + resultado surpreendente
- **"A maioria dos devs não sabe que..."** + dica técnica simples
- **Demos ao vivo** — mostrar o Claude Code ou Cursor fazendo algo em tempo real
- **Reação a notícias** — "o que significa para você que [grande empresa] fez X"
- **Desmistificação** — "Vibe Coding é realmente assim?" + teste honesto

### No YouTube (formato longo)
- Tutoriais completos de ferramentas novas
- "Construí X usando apenas Vibe Coding" (projetos reais)
- Entrevistas com devs sobre como estão usando AI no trabalho
- Análises críticas de pesquisas e estudos (como o da CodeRabbit)
